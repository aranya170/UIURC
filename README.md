# UIU Robotics Club Website

This is the official website of the UIU Robotics Club.

## Features

*   **Events:** View upcoming and past events.
*   **Event Registration:** Register for events through the website.
*   **Blog:** Read blog posts from the club members.
*   **Achievements:** View the club's achievements.
*   **About Us:** Learn more about the club and its members.

## Database Setup

1.  Create a new database in your MySQL server and name it `uiurc`.
2.  Import the `uiurc.sql` file into the `uiurc` database.
3.  The website is now connected to the database.
